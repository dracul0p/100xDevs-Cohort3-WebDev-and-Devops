Terminal
A terminal is a window where you type commands. It shows input and output. It only provides the interface.

Shell
A shell is the program that reads, interprets, and executes the commands you type in the terminal. Examples: Bash, Zsh, Fish.

Bash
Bash (Bourne Again Shell) is one type of shell. It is the most commonly used shell on Linux and macOS.
